#### 🔊 OpenSight
*Winner — Google Solution Challenge 2026, North America*

Blind users lose minutes to navigation on tasks that take seconds with sight. OpenSight completes them from one spoken sentence, eyes-free.

I built the shared session memory and the reasoning-based router that carries context across agents, so *"find me one under thirty dollars"* resolves against what you asked two steps ago. Also built the streaming voice pipeline.

Validated with the Blind Institute of Technology, Virginia DBVI, and the World Institute on Disability.
